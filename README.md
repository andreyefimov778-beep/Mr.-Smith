# Fabula.Smith — Фантастическая реальность

Архив невозможных теорий, альтернативных версий и чистой фантазии.

> ⚠️ Всё содержимое сайта является художественным вымыслом.

## Файлы сайта

```
fabula-smith/
├── index.html       — Главная страница с лентой теорий
├── categories.html  — Все категории теорий
├── disclaimer.html  — Отказ от ответственности
├── style.css        — Стили (sci-fi тема)
├── main.js          — Анимации (звёздное поле, появление карточек)
└── README.md        — Эта инструкция
```

## Как загрузить на GitHub Pages

### Шаг 1 — Загрузите файлы в репозиторий

1. Откройте ваш репозиторий на GitHub
2. Нажмите **Add file → Upload files**
3. Перетащите все файлы: `index.html`, `categories.html`, `disclaimer.html`, `style.css`, `main.js`
4. Нажмите **Commit changes**

### Шаг 2 — Включите GitHub Pages

1. Зайдите в **Settings** репозитория
2. В боковом меню выберите **Pages**
3. В разделе **Source** выберите ветку `main` и папку `/ (root)`
4. Нажмите **Save**

### Шаг 3 — Готово!

Через 1-2 минуты сайт будет доступен по адресу:
```
https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПОЗИТОРИЯ/
```

## Как добавить новую теорию

Откройте `index.html` и скопируйте блок `<article class="theory-card">...</article>`,
заменив тег, заголовок, текст и дату. Доступные теги:

- `tag-space` — Космос
- `tag-history` — История  
- `tag-tech` — Технологии
- `tag-paranormal` — Паранормальное
- `tag-conspiracy` — Конспирология
- `tag-nature` — Природа
